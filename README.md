
# Todo List App

This is a simple Todo List App built with React and Vite.

The app shows a title and a list of todos on the page. This project was created as part of my week 1 assignment

## Features

- Built with React
- Created using Vite
- Displays a list of todo items
- Beginner-friendly project structure

## Installation

1. Clone the repository:
   git clone https://github.com/your-username/todo-list.git
2. Go into the project folder:
   cd todo-list
3. Install dependencies:
    npm install
    Run the Development Server
    Start the app with:

    npm run dev

After running this command, open the local URL shown in the terminal:

http://localhost:5173



# React Todo Application

## Overview

This is a full-featured Todo Application built with React. The application allows users to manage tasks through creating, editing, completing, deleting, filtering, and sorting todos. It also includes user authentication and protected routes.

## Features

* User Authentication
* Create new todos
* Edit existing todos
* Mark todos as complete and incomplete
* Delete todos
* Search todos by title
* Filter todos (All, Active, Completed)
* Sort todos by creation date
* Protected routes using React Router
* Error handling and user feedback
* Responsive and user-friendly interface

## Technologies Used

### Frontend

* React
* React Router
* Context API
* useReducer
* JavaScript (ES6+)
* CSS
* Vite

### Backend Integration

* Fetch API
* CSRF Token Authentication

## Project Structure

* React Router for navigation
* Context API for authentication state management
* useReducer for todo state management
* Custom Hooks for reusable functionality
* Components organized by feature

## Security Considerations

* User input validation prevents empty todo submissions
* CSRF token protection for API requests
* Protected routes require authentication
* Error handling for failed API requests
* No sensitive credentials stored in client-side code
